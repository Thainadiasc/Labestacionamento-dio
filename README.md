# Sistema de Estacionamento - DIO

Projeto desenvolvido para o desafio da Trilha .NET - Fundamentos da Digital Innovation One (DIO).

## Descrição

Sistema simples para gerenciar veículos em um estacionamento, permitindo:

- Cadastrar veículos pela placa.
- Remover veículos e calcular o valor a ser pago conforme tempo estacionado.
- Listar veículos atualmente estacionados.

## Tecnologias

- Linguagem: C#
- Conceitos: listas, classes, métodos, encapsulamento, manipulação de console.

## Como usar

1. Clone ou baixe o repositório.
2. Abra o projeto em um ambiente C# (Visual Studio, VS Code, etc).
3. Compile e execute.
4. Utilize o menu para cadastrar, remover e listar veículos.
